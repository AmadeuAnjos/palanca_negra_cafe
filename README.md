# Palanca Negra Café

## Configuração necessária

- MySQL rodando localmente
- Banco existente: `palanca_negra_cafe`
- Usuário: `root` (sem senha)

## Como executar

### Frontend
```bash
cd frontend
npm install
npm run dev
```

### Backend
```bash
cd backend
npm install
node server.js
```

## Endpoints

- Teste de conexão: http://localhost:5000/api/test
- Frontend: http://localhost:3000
